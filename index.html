Refatoração da Biblioteca Axios

Projeto Acadêmico - Qualidade de Software e Governança

Este repositório apresenta a refatoração de um exemplo baseado na biblioteca Axios, com foco na melhoria da qualidade do código por meio de análise estática e boas práticas de desenvolvimento.

🔍 Objetivo

Aplicar conceitos de análise estática e refatoração em um projeto real para aprimorar:

  • Manutenibilidade
  • Legibilidade
  • Segurança
  • Desempenho

🛠️ Melhorias Aplicadas

  • Uso de const ao invés de var para imutabilidade
  • Uso de async/await para clareza do fluxo assíncrono
  • Tratamento de exceções mais detalhado com console.error
  • Remoção de lógica desnecessária

📁 Estrutura do Projeto

axios-refatorado/
├── original.js       # Versão com más práticas
├── refatorado.js     # Código com melhorias aplicadas
└── index.html        # Página inicial para GitHub Pages

🚀 Como Executar

git clone https://github.com/MariaEduarda/axios-refatorado.git
cd axios-refatorado
node refatorado.js

Ou visualize a refatoração diretamente aqui nesta página.
